📐 参数方程

理论齿廓（短幅外摆线）

x₀(θ) = R·cos(θ) − A·cos(Z₁·θ)
y₀(θ) = R·sin(θ) − A·sin(Z₁·θ)
θ ∈ [0, 2π]，Z₁ = Z₂ + 1

实际齿廓（等距曲线）

/* 法向量偏移 r */
dx₀/dθ = −R·sin(θ) + A·Z₁·sin(Z₁·θ)
dy₀/dθ = R·cos(θ) − A·Z₁·cos(Z₁·θ)
|T| = √(dx₀² + dy₀²)
x(θ) = x₀ − r·dy₀ / |T|
y(θ) = y₀ + r·dx₀ / |T|

运动学方程

/* 偏心角 φ 驱动 */
齿轮中心 = (A·cosφ, A·sinφ)
自转角 α = −φ / Z₂
/* 齿廓变换到世界坐标 */
x_w = x₀·cosα − y₀·sinα + A·cosφ
y_w = x₀·sinα + y₀·cosα + A·sinφ
